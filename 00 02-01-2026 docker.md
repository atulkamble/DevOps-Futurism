## ðŸ”¹ 8. Build, Tag, Push & Run Docker Image

### Clone Repository

```bash

cd pythonhelloworldtest
```

---

### Build Image

```bash
sudo docker buildx build -t docker.io/<your-dockerhub-username>/pythonhelloworld --load .
```

---

### Verify Image

```bash
sudo docker images
```

---

### Push Image to Docker Hub

```bash
sudo docker push docker.io/<your-dockerhub-username>/pythonhelloworld
```

---

### Run Image

```bash
sudo docker run docker.io/<your-dockerhub-username>/pythonhelloworld
```

---

### Pull & Run Existing Image

```bash
sudo docker pull atuljkamble/pythonhelloworld
sudo docker run atuljkamble/pythonhelloworld
```

## ðŸ”¹ 9. Install Docker Compose (Linux)
